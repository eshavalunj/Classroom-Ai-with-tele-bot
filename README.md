# GradeSync - AI-Powered Student Management System

## 🚀 Overview

GradeSync is a modern, AI-enhanced student management system designed for the 2026 AI-driven job market. This project demonstrates cutting-edge web development skills combined with artificial intelligence integration, making it a standout portfolio piece for developers seeking roles in AI-first companies.

## ✨ Key Features

### 🤖 AI-Powered Features
- **AI Chat Assistant**: Integrated conversational AI for instant help with student management queries
- **Personalized AI Insights**: Generate tailored feedback and study recommendations for each student based on their performance
- **Intelligent Analytics**: AI-driven grade analysis and predictive insights

### 🎨 Modern UI/UX
- **Dark/Light Theme Toggle**: Seamless theme switching with persistent preferences
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Smooth Animations**: Professional micro-interactions and transitions

### 📊 Core Functionality
- **CRUD Operations**: Complete Create, Read, Update, Delete for student records
- **Real-time Statistics**: Live dashboard with total students, average GPA, and pass rates
- **Advanced Search**: Instant filtering by name or roll number
- **Data Export**: CSV export functionality for data portability
- **Local Storage**: Persistent data storage without backend requirements

## 🛠️ Technology Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **AI Integration**: OpenAI GPT API (configurable)
- **Styling**: Custom CSS with CSS Variables for theming
- **Storage**: Browser Local Storage
- **Icons**: Font Awesome 6

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/gradesync.git
   cd gradesync
   ```

2. **Open in browser**
   - Simply open `index.html` in any modern web browser
   - No build process or dependencies required

3. **Configure AI Features** (Optional)
   - Obtain an OpenAI API key
   - Replace `YOUR_OPENAI_API_KEY` in `script.js` with your actual key
   - Uncomment the API integration code for full AI functionality

## 📈 Usage

1. **Add Students**: Click "Add Student" to input student information
2. **View Analytics**: Check the header stats for real-time insights
3. **AI Assistance**: Click the chat icon for AI-powered help
4. **Generate Insights**: Use "Get Insights" for personalized AI feedback
5. **Export Data**: Download student data as CSV
